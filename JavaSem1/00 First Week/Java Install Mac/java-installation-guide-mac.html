Java Installation Guide for Mac

Introduction

This is a short bit of information on using the installation file provided on Dr. Neat’s website. The file runs a series of commands at the terminal; if you would like to know what these are, or want to do them yourself(which would be a good introductory exercise for using the terminal!), visit the about-installing-java page on Dr. Neat’s website.

Integration with Macs is fairly new and experimental for the time being, so be sure to report anything weird or unexpected that happens!

Run the script provided on Dr. Neat’s Website called install-java.command. Double clicking it should work fine. This should open a terminal that will start to run a series of commands. It will likely prompt you for your password; when it does so, enter the password and press enter.
